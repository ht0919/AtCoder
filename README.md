## 環境

- OS：Windows/macOS/Linux
- 言語：C++

## 操作手順

- ビルド：g++ file.cpp
- 実行(Windows)：a.exe < test[1|2|3|4].txt
- 実行(macOS/Linux)：./a.out < test[1|2|3|4].txt
